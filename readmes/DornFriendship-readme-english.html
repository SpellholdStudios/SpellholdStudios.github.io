Introduction


One of the things many find sorely lacking in the Bioware / Beamdog NPCs are the more complex relationships presented by modern NPC mods--especially in regards to "friendships", the relationships NPCs have with the PC when not romancing them.

This mod adds a series of dialogues with Dorn.

Visit the website or forum for all the latest updates.

Installation


Windows

The Dorn Friendship is a WeiDU mod, packaged in an extractable .ZIP archive. Simply unpack the mod package into your Baldur's Gate II EE directory and run "setup-DornFriendship.exe".

To un-install, run "setup-DornFriendship.exe" again and select the un-install option.



Compatibility and Installation Order

The Dorn Friendship mod should be compatible with all WeiDU mods, however, we cannot test every single one. It is only compatible with BG2EE. If you do encounter an error, please let us know by posting the message at our forums.

Frequently Asked Questions


Q: Is the Dorn Friendship compatible with XYZ mod?
A: It most certainly should be, but we have not tested it with every single mod available.

Q: When will the friendship start?
A: After you join Dorn. His original relationship should start soon. Once you are given a chance to sleep with him, reject him and suggest you are not interested in the romance. You must do it early in the game. If you "kill" his original romance path that way and do not sleep with him, the friendship talks should kick in. The mod also includes some scenery talks.

Q: This mod isn't in character at *all*.
A: We're always looking for ways to improve the characterization in this mod, so please post any suggestions or constructive criticism on the forums.

Q: Have you got any plans to expand the mod by adding more dialogues in the future?
A: There are no current plans to add to the mod, no.

Q: How many dialogues are there added in total?
A: With this mod installed, you should be able to experience up to 15 talks in SoA (6 timered talks and 9 scenery / situational talks) and up to 7 dialogues in ToB (3 timered and 4 scenery / situational talks - 3 of these will be skipped if they already happened in SoA portion of the game).

Q: Does this mod change the game in any way besides adding more dialogue to Dorn?
A: This mod also includes a small fix for his original relationship path which partially affects the new friendship path. Once the bug is fixed by a patch, then this fix will be disabled and only talks will be installed.

Q: Is there any chance that player-initiated dialogue will be added in the future?
A: There are currently no plans for it.

Q: I would like to translate the Dorn Friendship mod into another language. Any chance you could TRAify the mod?
A: Sure, we'd be happy to. If you are interested in translating, just send us a line and I'll see what I can do for the next version.
